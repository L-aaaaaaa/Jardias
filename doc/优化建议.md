这段代码的核心问题是 **`collect_round` 这个函数承担了太多职责**，导致它成了一个超过 200 行的"上帝函数"，内部状态变量多达 20 余个，嵌套深度超过 4 层，重复逻辑散布多处。以下是我从 **架构、代码质量、可维护性** 三个维度给出的优化建议，并附带了关键重构示例。

---

## 一、架构层面：拆分巨型函数

当前 `collect_round` 同时在做：
1. 流式 chunk 消费
2. 多源推理内容解析（DeepSeek / MiniMax / `<think>`）
3. 行级去重与缓冲
4. 工具调用收集
5. 诊断日志输出

**优化方向**：将 `collect_round` 拆分为一个 **状态收集器类** + 若干 **纯函数处理器**。

```python
from enum import Enum, auto
from dataclasses import dataclass, field

class ReasoningSource(Enum):

class StreamChunk(Protocol):
    choices: list
    usage: object | None

    reasoning_parts: list[str] = field(default_factory=list)
```

## 二、消除重复逻辑：提取行级去重

代码中**行级去重逻辑重复了 5 次**（think 内、reasoning_details 内、content 缓冲内等），几乎一模一样：

```python
# 重复模式（出现 5 次）
_new_think = []
for line in dc.split("\n"):
    stripped = line.rstrip()
    if stripped and stripped not in _rd_printed_lines:
        _rd_printed_lines.append(stripped)
        _new_think.append(line)
if _new_think:
    stream_print("\n".join(_new_think))
```

**优化为单一工具函数**：

```python
class LineDeduplicator:
    def __init__(self):
        self._seen: set[str] = set()
    
    def filter_new(self, text: str) -> str:
        """返回去重后的新文本，并更新已见集合"""
        new_lines = []
        for line in text.split("\n"):
            stripped = line.rstrip()
            if stripped and stripped not in self._seen:
                self._seen.add(stripped)
                new_lines.append(line)
        return "\n".join(new_lines) if new_lines else ""
```

## 三、改善 `<think>` 标签解析

当前用 `in` 和 `split` 手动解析 `<think>` 标签，**如果标签被 chunk 边界切分（如 `<` 在 chunk N，`think>` 在 chunk N+1），会完全失效**。

**优化**：使用状态机缓冲，不依赖单次 chunk 的完整性：

```python
@dataclass
class ThinkTagParser:
    """可靠解析跨 chunk 的 <think>...</think>"""
    _buffer: str = ""
    _in_think: bool = False
    _think_content: list[str] = field(default_factory=list)
    
    THINK_OPEN = "<think>"
    THINK_CLOSE = "</think>"
    
    def feed(self, text: str) -> tuple[str, str | None]:
        """
        返回: (content_text, think_text)
        think_text 为 None 表示 think 尚未闭合
        """
        self._buffer += text
        
        if not self._in_think:
            if self.THINK_OPEN in self._buffer:
                pre, post = self._buffer.split(self.THINK_OPEN, 1)
                self._buffer = post
                self._in_think = True
                if pre:
                    return pre, None  # think 刚开始，content 部分
                return "", None
            else:
                # 检查尾部是否可能是 <think> 的前缀（如 "<thi"）
                # 简单处理：保留最后 6 个字符作为缓冲
                if len(self._buffer) > 6:
                    emit = self._buffer[:-6]
                    self._buffer = self._buffer[-6:]
                    return emit, None
                return "", None
        else:
            if self.THINK_CLOSE in self._buffer:
                pre, post = self._buffer.split(self.THINK_CLOSE, 1)
                self._think_content.append(pre)
                self._in_think = False
                self._buffer = post
                think = "".join(self._think_content)
                self._think_content.clear()
                return post, think
            else:
                # 检查尾部是否可能是 </think> 前缀
                if len(self._buffer) > 7:
                    emit = self._buffer[:-7]
                    self._buffer = self._buffer[-7:]
                    self._think_content.append(emit)
                return "", None
```

## 四、使用枚举与策略模式替代魔法字符串

当前用字符串比较区分推理来源：

```python
if reasoning_field == "reasoning_content":
    ...
elif reasoning_field == "reasoning_details":
    ...
```

**优化**：

```python
class ProviderFormat(Enum):
    DEEPSEEK = "reasoning_content"
    MINIMAX = "reasoning_details"
    DASHSCOPE = "reasoning_content"  # 与 DeepSeek 相同

class ReasoningExtractor(Protocol):
    def extract(self, delta) -> str | None: ...

class DeepSeekExtractor:
    def __init__(self):
        self._last = ""
    def extract(self, delta) -> str | None:
        rc = getattr(delta, "reasoning_content", None)
        if not rc:
            return None
        # 累积式 diff
        if rc.startswith(self._last):
            new = rc[len(self._last):]
            self._last = rc
            return new
        self._last += rc
        return rc

class MiniMaxExtractor:
    def __init__(self):
        self._last = ""
        self._dedup = LineDeduplicator()
    def extract(self, delta) -> str | None:
        details = getattr(delta, "reasoning_details", None)
        if not details:
            return None
        texts = []
        for d in details:
            text = d.get("text", d.get("content", "")) if isinstance(d, dict) else str(d)
            if text:
                if text.startswith(self._last):
                    new = text[len(self._last):]
                    self._last = text
                    texts.append(new)
                else:
                    texts.append(text)
                    self._last = text
        merged = "\n".join(texts)
        return self._dedup.filter_new(merged)
```

## 五、改善类型注解与数据结构

**总结**：这段代码的优化核心就一句话——**"把 `collect_round` 里那 20 个闭包变量和 5 套重复的去重逻辑，收进一个类里，再把不同 provider 的解析逻辑拆成策略对象"**。这样代码行数能减少 30% 以上，测试覆盖率能从"几乎无法测"变成"每个策略可独立测"。

```
import json
import re as _re

from .reply_getter import get_ipu_stream_reply  # noqa: F401

MAX_ITER = 999


# ============================================================================
# 0. 类型定义
# ============================================================================

class Message(TypedDict):
    role: str
    content: str
    tool_calls: list[dict] | None
    tool_call_id: str | None
    name: str | None
    reasoning_content: str | None
    _reasoning: bool | None


# ============================================================================
# 1. 行级去重（所有推理源共享同一实例）
# ============================================================================

class LineDeduplicator:
    """全局行级去重。同一行文本不会在 reasoning_content / reasoning_details / <think> 之间重复输出。"""

    def __init__(self):
        self._seen: set[str] = set()

    def filter_new(self, text: str) -> str:
        if not text:
            return ""
        new_lines: list[str] = []
        for line in text.split("\n"):
            stripped = line.rstrip()
            if stripped and stripped not in self._seen:
                self._seen.add(stripped)
                new_lines.append(line)
        return "\n".join(new_lines) if new_lines else ""


# ============================================================================
# 2. 提取器（无继承，纯实现类；调度靠字典注册表）
# ============================================================================

class DeepSeekExtractor:
    """DeepSeek / DashScope: reasoning_content 字段，累积全量。"""

    def __init__(self, dedup: LineDeduplicator):
        self._dedup = dedup
        self._last = ""
        self._parts: list[str] = []
        self._produced = False

    def extract(self, delta: Any) -> str | None:
        rc = getattr(delta, "reasoning_content", None)
        if not rc:
            return None

        self._produced = True
        # 累积式 diff
        if rc.startswith(self._last):
            new_text = rc[len(self._last) :]
            self._last = rc
            if new_text:
                filtered = self._dedup.filter_new(new_text)
                if filtered:
                    self._parts.append(filtered)
                return filtered
            return None

        # 非累积式：每个 delta 仅含新分块
        self._parts.append(rc)
        self._last += rc
        return self._dedup.filter_new(rc)

    def get_cumulative(self) -> str:
        from_parts = "".join(self._parts)
        if len(self._last) < len(from_parts):
            return from_parts
        return self._last or from_parts

    def has_produced(self) -> bool:
        return self._produced


class MiniMaxExtractor:
    """MiniMax: reasoning_details 列表，每项含 text / content。"""

    def __init__(self, dedup: LineDeduplicator):
        self._dedup = dedup
        self._last = ""
        self._parts: list[str] = []
        self._produced = False

    def extract(self, delta: Any) -> str | None:
        details = getattr(delta, "reasoning_details", None)
        if not details:
            return None

        self._produced = True
        texts: list[str] = []

        for d in details:
            text = d.get("text", d.get("content", "")) if isinstance(d, dict) else str(d)
            if not text:
                continue

            if text.startswith(self._last):
                new_text = text[len(self._last) :]
                self._last = text
                if new_text:
                    filtered = self._dedup.filter_new(new_text)
                    if filtered:
                        texts.append(filtered)
                        self._parts.append(filtered)
            else:
                filtered = self._dedup.filter_new(text)
                if filtered:
                    texts.append(filtered)
                    self._parts.append(filtered)
                self._last = text

        return "\n".join(texts) if texts else None

    def get_cumulative(self) -> str:
        return self._last or "\n".join(self._parts)

    def has_produced(self) -> bool:
        return self._produced


# 字典策略表：reasoning_field -> 提取器工厂
EXTRACTOR_REGISTRY: dict[str, Callable[[LineDeduplicator], Any]] = {
    "reasoning_content": DeepSeekExtractor,
    "reasoning_details": MiniMaxExtractor,
}


def _create_extractor(reasoning_field: str, dedup: LineDeduplicator) -> Any:
    factory = EXTRACTOR_REGISTRY.get(reasoning_field)
    if factory is None:
        raise ValueError(f"未知的 reasoning_field: {reasoning_field!r}")
    return factory(dedup)

class OutputHandler:

# ============================================================================
# 4. 单轮收集器（替代 collect_round 的 20+ 闭包变量）
# ============================================================================

@dataclass
class RoundCollector:
    """消费流式响应，边接收边输出，同时返回结构化 RoundOutput。"""

    reasoning_field: str = "reasoning_details"
    is_tool_round: bool = False
    output_handler: OutputHandler = field(default_factory=lambda: OutputHandler())

    # 组件
    _dedup: LineDeduplicator = field(default_factory=LineDeduplicator)
    _extractor: Any = field(init=False)

    # 结果收集
    reasoning_parts: list[str] = field(default_factory=list)
    content_parts: list[str] = field(default_factory=list)
    tool_names: list[str] = field(default_factory=list)
    tool_args: list[str] = field(default_factory=list)

    # think 标签状态机（跨 chunk 安全）
    _stream_buffer: str = ""
    _in_think: bool = False
    _think_acc: str = ""

    # MiniMax 推理泄漏缓冲
    _content_buffer: list[str] = field(default_factory=list)
    _waiting_reasoning: bool = field(init=False)

    # 推理主源
    _primary_source: str | None = None  # "field" | "think"

    # 元数据
    finish_reason: str | None = None
    usage: dict | None = None
    deltas: list = field(default_factory=list)

    def __post_init__(self):
        self._waiting_reasoning = self.reasoning_field == "reasoning_details"
        self._extractor = _create_extractor(self.reasoning_field, self._dedup)

    # ------------------------------------------------------------------
    # 公共入口
    # ------------------------------------------------------------------
    def consume(self, stream) -> RoundOutput:
        for chunk in stream:
            self._handle_chunk(chunk)
            if self.finish_reason == "tool_calls":
                break

        reasoning = self._finalize_reasoning()
        content = self._finalize_content()
        calls = [ToolCall(n, a) for n, a in zip(self.tool_names, self.tool_args) if n]

        return RoundOutput(
            reasoning, content, calls, self.deltas,
            finish_reason=self.finish_reason, usage=self.usage,
        )

    # ------------------------------------------------------------------
    # Chunk 分发
    # ------------------------------------------------------------------
    def _handle_chunk(self, chunk: Any):
        if not getattr(chunk, "choices", None):
            if hasattr(chunk, "usage") and chunk.usage:
                self.usage = chunk.usage.model_dump() if hasattr(chunk.usage, "model_dump") else None
            return
        if not chunk.choices:
            return

        delta = chunk.choices[0].delta
        self.deltas.append(delta)

        self._handle_reasoning_field(delta)
        self._handle_content(delta)
        self._handle_tool_calls(delta)

        if hasattr(chunk, "usage") and chunk.usage:
            self.usage = chunk.usage.model_dump() if hasattr(chunk.usage, "model_dump") else None
        self.finish_reason = chunk.choices[0].finish_reason if chunk.choices else None

    # ------------------------------------------------------------------
    # 专用字段推理
    # ------------------------------------------------------------------
    def _handle_reasoning_field(self, delta: Any):
        new_text = self._extractor.extract(delta)
        if new_text is None:
            return

        if self._primary_source is None:
            self._primary_source = "field"
            self.output_handler.show_reasoning_header()

        if self._primary_source == "field":
            self.reasoning_parts.append(new_text)
            self.output_handler.print_reasoning(new_text)
            self._flush_content_buffer()

    # ------------------------------------------------------------------
    # Content（含跨 chunk 安全的 <think> 解析）
    # ------------------------------------------------------------------
    def _handle_content(self, delta: Any):
        dc = getattr(delta, "content", None)
        if not dc:
            return

        self._stream_buffer += dc

        while self._stream_buffer:
            if not self._in_think:
                idx = self._stream_buffer.find("<think>")
                if idx != -1:
                    pre = self._stream_buffer[:idx]
                    self._stream_buffer = self._stream_buffer[idx + 7 :]
                    if pre:
                        self._emit_content(pre)
                    self._in_think = True
                    self._think_acc = ""
                    if self._primary_source is None:
                        self._primary_source = "think"
                        self.output_handler.show_reasoning_header()
                    continue
                else:
                    # 保留可能是 "<think>" 前缀的尾部
                    keep = 6
                    if len(self._stream_buffer) > keep:
                        emit = self._stream_buffer[:-keep]
                        self._stream_buffer = self._stream_buffer[-keep:]
                        self._emit_content(emit)
                        continue
                    break
            else:
                idx = self._stream_buffer.find("</think>")
                if idx != -1:
                    pre = self._stream_buffer[:idx]
                    post = self._stream_buffer[idx + 8 :]
                    self._stream_buffer = post
                    self._in_think = False
                    if pre:
                        self._think_acc += pre
                        self._emit_think(self._think_acc, final=True)
                    self._think_acc = ""
                    if post:
                        self._emit_content(post)
                    continue
                else:
                    # 保留可能是 "</think>" 前缀的尾部
                    keep = 7
                    if len(self._stream_buffer) > keep:
                        emit = self._stream_buffer[:-keep]
                        self._stream_buffer = self._stream_buffer[-keep:]
                        self._think_acc += emit
                        self._emit_think(emit, final=False)
                        continue
                    break

    # ------------------------------------------------------------------
    # 工具调用收集
    # ------------------------------------------------------------------
    def _handle_tool_calls(self, delta: Any):
        tcs = getattr(delta, "tool_calls", None)
        if not tcs:
            return

        for tc_d in tcs:
            idx = tc_d.index
            while idx >= len(self.tool_names):
                self.tool_names.append("")
                self.tool_args.append("")

            name = getattr(tc_d.function, "name", None) or ""
            args = getattr(tc_d.function, "arguments", None) or ""
            if name:
                self.tool_names[idx] = name
            if args:
                self.tool_args[idx] += args

    # ------------------------------------------------------------------
    # 内容发射
    # ------------------------------------------------------------------
    def _emit_content(self, text: str):
        if not text:
            return
        if self._waiting_reasoning and self._primary_source is None:
            self._content_buffer.append(text)
        else:
            self.output_handler.show_content_header()
            self.content_parts.append(text)
            self.output_handler.print_content(text)

    def _emit_think(self, text: str, final: bool):
        if self._primary_source != "think":
            return
        filtered = self._dedup.filter_new(text)
        if filtered:
            self.reasoning_parts.append(filtered)
            self.output_handler.print_reasoning(filtered)

    def _flush_content_buffer(self):
        if not self._content_buffer:
            return
        buf_text = "".join(self._content_buffer)
        self._content_buffer.clear()

        filtered = self._dedup.filter_new(buf_text)
        if filtered:
            self.output_handler.show_content_header()
            self.content_parts.append(filtered)
            self.output_handler.print_content(filtered)

    # ------------------------------------------------------------------
    # 最终化
    # ------------------------------------------------------------------
    def _finalize_reasoning(self) -> str:
        streamed = "".join(self.reasoning_parts)

        if self._extractor.has_produced():
            field_reasoning = self._extractor.get_cumulative()
            if len(field_reasoning) != len(streamed):
                logger.debug(
                    f"    [推理诊断] API累积={len(field_reasoning)}字 stream拼回={len(streamed)}字 "
                    f"差值={len(field_reasoning) - len(streamed)}"
                )
            return field_reasoning or streamed

        if self._in_think:
            tail = self._dedup.filter_new(self._think_acc + self._stream_buffer)
            if tail:
                return streamed + tail

        return streamed

    def _finalize_content(self) -> str:
        if self._content_buffer:
            buf_text = "".join(self._content_buffer)
            filtered = self._dedup.filter_new(buf_text)
            if filtered:
                self.content_parts.append(filtered)

        if self._stream_buffer and not self._in_think:
            self.content_parts.append(self._stream_buffer)

        content = "".join(self.content_parts)
        content = _re.sub(r"<think>.*?</think>\s*", "", content, flags=_re.DOTALL).strip()
        return content

async def reason_act_loop(
    messages: list[dict],
    ipu_config: Any,
    reasoning_field: str = "reasoning_details",
    reasoning_inline: bool = False,
    character_name: str = "",
    on_history_save: Callable | None = None,
) -> ChatResult:
    if character_name:
        set_display_name(character_name)

    for i in range(MAX_ITER):
        output, messages = await _run_single_round(
            messages, i, ipu_config,
            reasoning_field=reasoning_field,
            reasoning_inline=reasoning_inline,
            character_name=character_name,
            is_tool_round=(i > 0),
            on_history_save=on_history_save,
        )

        if output.tool_calls:
            _log_tool_calls(output.tool_calls)
            _display_tool_calls(output.tool_calls)
            messages = await _execute_tools(output.tool_calls, messages, i, on_history_save)

            switch = pop_switch()
            if switch:
                return ChatResult(
                    messages=messages,
                    should_switch=True,
                    switch_provider=switch.provider,
                    switch_ipu=switch.ipu,
                )
        else:
            round_end(i + 1, "no tool calls" if i == 0 else "tool chain done")
            return ChatResult(messages=messages)

    max_rounds_reached(MAX_ITER)
    return ChatResult(messages=messages)


async def _run_single_round(
    messages: list[dict],
    iteration: int,
    ipu_config: Any,
    reasoning_field: str = "reasoning_details",
    reasoning_inline: bool = False,
    character_name: str = "",
    is_tool_round: bool = False,
    on_history_save: Callable | None = None,
):
    round_start(iteration + 1, len(messages))

    stream = get_ipu_stream_reply(messages, ipu_config=ipu_config)
    t0 = time.time()

    collector = RoundCollector(
        reasoning_field=reasoning_field,
        is_tool_round=is_tool_round,
        output_handler=OutputHandler(silent=get_silent(), is_tool_round=is_tool_round),
    )
    output = collector.consume(stream)

    collector.output_handler.print_newline()
    if output.content.strip():
        collector.output_handler.print_end_separator()

    line = format_round_usage(output.usage)
    collector.output_handler.print_usage(line)

    elapsed = time.time() - t0
    logger.info(f"    {format_api_ok(elapsed, output.usage, output.finish_reason)}")
    set_round_meta(elapsed, output.usage, output.finish_reason)

    if not reasoning_inline and output.reasoning:
        messages.append({"role": "assistant", "content": output.reasoning, "_reasoning": True})

    msg: dict = {"role": "assistant", "content": output.content}
    if reasoning_inline and output.reasoning:
        msg["reasoning_content"] = output.reasoning
    if output.tool_calls:
        msg["tool_calls"] = [
            {"id": f"call_{iteration}_{i}", "type": "function",
             "function": {"name": tc.name, "arguments": tc.arguments}}
            for i, tc in enumerate(output.tool_calls)
        ]
    messages.append(msg)

    if on_history_save:
        on_history_save()

    return output, messages

def _log_tool_calls(tool_calls: list[ToolCall]):
    calls = []
    for tc in tool_calls:
        try:
            args = json.loads(tc.arguments) if isinstance(tc.arguments, str) else tc.arguments
            args_str = ", ".join(f"{k}={v}" for k, v in args.items())
        except Exception:
            args_str = tc.arguments[:80] if isinstance(tc.arguments, str) else str(tc.arguments)[:80]
        calls.append(f"{tc.name}({args_str})")
    logger.info(f"    [TOOL] 工具调用 | {len(tool_calls)} 个: {'; '.join(calls)}")


def _display_tool_calls(tool_calls: list[ToolCall]):
    if get_silent():
        return

    separate_print(title="工具调用")
    for tc in tool_calls:
        try:
            args = json.loads(tc.arguments) if isinstance(tc.arguments, str) else tc.arguments
        except Exception:
            args = {}

        if tc.name == "send_to_character":
            recipient = args.get("recipient", "?")
            message = args.get("message", "")
            print(f"  >> {tc.name} → {recipient}:")
            print(f"     {message}")
        elif tc.name == "shice_schedule_add":
            desc = args.get("message", args.get("description", "")) or "?"
            timestamps = args.get("timestamps", [])
            count = len(timestamps) if isinstance(timestamps, list) else "?"
            print(f"  >> {tc.name}: {desc[:60]}（{count} 个时间点）")
        else:
            print(f"  >> {tc.name}")


async def _execute_tools(
    tool_calls: list[ToolCall],
    messages: list[dict],
    round_idx: int,
    on_history_save: Callable | None = None,
) -> list[dict]:
    from tool.builtin import tools

    for idx, tc in enumerate(tool_calls):
        try:
            args = json.loads(tc.arguments) if isinstance(tc.arguments, str) else tc.arguments
            result = await tools.execute(tc.name, args)
        except Exception as e:
            result = f"[Error] {type(e).__name__}: {e}"

        _log_tool_result(tc.name, result)
        _display_tool_result(tc.name, result)

        messages.append({
            "role": "tool",
            "tool_call_id": f"call_{round_idx}_{idx}",
            "name": tc.name,
            "content": result,
        })

        if on_history_save:
            on_history_save()

        if tc.name == "send_to_character":
            messages.append({
                "role": "user",
                "content": (
                    "[系统] send_to_character 已完成。"
                    "对方角色的回复在上方 tool result 中。"
                    "⚠️ 对方无法看到你的普通回复文本。"
                    "如需继续对话 → 调用 send_to_character。"
                    "如需向用户汇报 → 直接输出回复。"
                ),
            })

    return messages


def _log_tool_result(tool_name: str, result: str):
    if tool_name == "send_to_character":
        lines = result.split("\n")
        brief = "\n".join(lines[:2])
        if len(brief) > 300:
            brief = brief[:300] + "..."
        logger.info(f"    [RESULT] 工具结果 | {tool_name} →\n{brief}")
    else:
        first_line = result.split("\n")[0]
        if len(first_line) > 120:
            first_line = first_line[:120] + "..."
        logger.info(f"    [RESULT] 工具结果 | {tool_name} → {first_line}")


def _display_tool_result(tool_name: str, result: str):
    if get_silent():
        return

    if tool_name == "send_to_character":
        separate_print(title=f"{tool_name} 回复")
        try:
            print(f"  {result}")
        except UnicodeEncodeError:
            safe = result.encode("ascii", errors="ignore").decode("ascii")
            print(f"  [内容已简化] {safe[:500]}")
    elif tool_name == "shice_schedule_add":
        print(f"  [OK] {tool_name}: {result[:200]}")
    else:
        print(f"\n  [OK] {tool_name}:\n{result[:300]}{'...' if len(result) > 300 else ''}")
```
